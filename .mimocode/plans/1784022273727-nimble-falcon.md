### `paste_entry()` 算法

```
1. target = LAST_ACTIVE_HWND.load()
2. 将内容写入剪贴板 (set_clipboard_text)
3. 设置 LAST_APP_SET_HASH 防回声
4. CLIPBOARD_MONITOR_PAUSED = true
5. sleep 50ms
6. SetForegroundWindow(target)
7. sleep 50ms
8. SendInput: Ctrl+V keydown/keyup（扫描码）
9. sleep 100ms
10. CLIPBOARD_MONITOR_PAUSED = false
11. 隐藏窗口
```

### `hotkey.rs` 架构

- 专用线程 + Win32 消息泵 (GetMessageW/DispatchMessageW)
- `RegisterHotKey(hwnd, id, MOD_CONTROL | MOD_SHIFT, VK_V)` 注册 Ctrl+Shift+V
- 收到 `WM_HOTKEY` → 调用 `window_manager::toggle_window()`
- `parse_hotkey_for_hook()` 解析 "Ctrl+Shift+V" 格式的字符串

### `window_manager.rs` 核心逻辑

- **显示**: `WS_EX_NOACTIVATE` 设置 → ShowWindow(SW_SHOWNA) → SetWindowPos(HWND_TOPMOST, SWP_NOACTIVATE)
- **聚焦搜索**: 移除 `WS_EX_NOACTIVATE` → force_focus_window (AttachThreadInput + SetForegroundWindow)
- **隐藏**: release_win_keys → hide → restore_last_focus (回到 LAST_ACTIVE_HWND)
- **Toggle**: 检查 IS_HIDDEN 原子，调用 show 或 hide

### 验收标准
- [ ] Ctrl+Shift+V 显示窗口，再次按下隐藏
- [ ] 在记事本中复制文字，点击条目 → 粘贴到记事本
- [ ] 粘贴操作不产生回声条目
- [ ] 窗口不抢焦点 (WS_EX_NOACTIVATE)

## 注意事项

1. **windows-reactor 约束**: 子组件使用元组而非Vec，如 `vstack((child1, child2, child3))`
2. **Edition 2024**: `#[macro_use]` 已弃用，使用显式 `use crate::{info, error};`
3. **HWND as usize**: 跨 async 边界传递 HWND 时使用 `usize` 避免 `!Send`
4. **IME回声标记**: 注入的按键使用 `dwExtraInfo = 0x57494E50` ("WINP") 标记
5. **WS_EX_NOACTIVATE**: 显示时设置，聚焦搜索时移除，隐藏时恢复
6. **防回声**: 粘贴前设置 LAST_APP_SET_HASH，监控器跳过匹配的哈希
7. **扫描码优先**: SendInput 使用 KEYEVENTF_SCANCODE 而非虚拟键，兼容性更好
